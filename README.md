# zantarix/actions

GitHub Actions published by Zantarix. Each action lives in its own subdirectory and is referenced as `uses: zantarix/actions/<name>@<sha>`.

## Actions

| Action | Description |
|--------|-------------|
| [setup-cursus](setup-cursus/README.md) | Installs the [cursus](https://github.com/zantarix/cursus) release-management CLI, verifies its provenance, and adds it to `PATH`. |
